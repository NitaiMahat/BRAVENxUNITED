# Fly Forward: Early Aviation Mentorship

A prototype website for an early aviation mentorship program designed to give middle-school students of color exposure to aviation through relatable mentors and hands-on learning.

## Project Structure

```
BravenPrototype/
├── index.html          # Main HTML file
├── css/
│   └── styles.css      # Custom CSS styles and animations
├── js/
│   └── main.js         # JavaScript for interactions and animations
├── assets/             # Images and other static assets (future use)
└── README.md           # Project documentation
```

## Features

- **Modern Design**: Clean, responsive layout using TailwindCSS
- **Smooth Animations**: Fade-in effects on scroll using Intersection Observer API
- **Interactive Elements**: Hover effects and smooth scrolling navigation
- **Color Palette**:
  - Sky Blue: `#4FC3F7`
  - White: `#FFFFFF`
  - Soft Gray: `#F8FAFC`
  - Accent Orange: `#FB923C`

## Sections

1. **Header**: Sticky navigation with program name and links
2. **Hero Section**: Bold headline with call-to-action button
3. **About the Program**: Three feature cards highlighting program benefits
4. **Program Flow**: Four-week timeline visualization
5. **Mentor Spotlight**: Featured mentor profile
6. **Sign-Up Section**: Application form (prototype - no backend)
7. **Footer**: Copyright and contact information

## Technologies Used

- **HTML5**: Semantic markup
- **TailwindCSS**: Utility-first CSS framework (via CDN)
- **Vanilla JavaScript**: No dependencies, pure ES6+
- **Intersection Observer API**: For scroll animations

## Getting Started

1. Open `index.html` in a modern web browser
2. No build process or dependencies required
3. For development, use a local server (optional):
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (with http-server)
   npx http-server
   ```

## Notes

- This is a **prototype** website, not a production application
- Form submissions are handled client-side only (no backend)
- Images are sourced from Unsplash (placeholder URLs)
- All animations are CSS-based for optimal performance
